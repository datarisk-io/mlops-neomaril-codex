==============
Neomaril-Codex
==============

Package for interacting with Neomaril, a tool for deploying ML models.


How to generate a package
-------------------------

Install pipenv

.. code:: python

   pip install pipenv

Install the package

.. code:: python
   
   pipenv install
   pipenv shell
