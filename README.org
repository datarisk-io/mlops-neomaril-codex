#+TITLE: Neomaril Codex

* About
Package for interacting with [[https://github.com/datarisk-io/neomaril][Neomaril]], a tool for deploying ML models.

** How to generate a package

Install pipenv
#+BEGIN_SRC shell
  pip install pipenv
#+END_SRC

Install the package
#+BEGIN_SRC shell
  pipenv install
  pipenv shell
#+END_SRC
