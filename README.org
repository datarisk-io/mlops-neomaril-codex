#+TITLE: Neomaril Codex

* About
Package for interacting with [[https://gitlab.com/datarisk-dev/product/mlops/proof-of-concept/neomaril][Neomaril]], a tool for deploying ML models.

** How to generate a package

Install poetry
#+BEGIN_SRC shell
  pip install poetry
#+END_SRC

Install the package
#+BEGIN_SRC shell
  poetry shell
  poetry install
#+END_SRC
