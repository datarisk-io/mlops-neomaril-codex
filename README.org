#+TITLE: Neomaril Codex

* About
Package for interacting with [[https://github.com/datarisk-io/neomaril][Neomaril]], a tool for deploying ML models.

** For developers

Install pipenv
#+BEGIN_SRC shell
  pip install pipenv
#+END_SRC

Install the package enviroment
#+BEGIN_SRC shell
  pipenv update --dev
  pipenv shell
#+END_SRC
